Ответы на вопросы для самоконтроля

1. Основные различия между development и production:
   - Development: подробные логи, горячая перезагрузка, отладочная информация
   - Production: минимизированный код, оптимизация, безопасные настройки

2. Важность environment variables в production:
   - Безопасность: скрытие API ключей, паролей БД
   - Гибкость: разные настройки для разных сред
   - Не нужно менять код при смене конфигурации

3. Управление разными средами:
   - Использовать разные .env файлы
   - Настройки в CI/CD пайплайнах
   - Флаги функциональности (feature flags)

4. Методы оптимизации производительности:
   - Минификация кода
   - Сжатие Gzip/Brotli
   - Кэширование
   - CDN для статических файлов

5. Мониторинг ошибок в production:
   - Сервисы типа Sentry, LogRocket
   - Логи на сервере
   - Метрики производительности

6. CI/CD и преимущества:
   - Автоматизация тестирования и деплоя
   - Быстрые и безопасные обновления
   - Постоянная доступность приложения

7. Обновление без downtime:
   - Blue-green деплой
   - Canary релизы
   - Docker с оркестрацией

8. Меры безопасности при деплое:
   - HTTPS везде
   - Защита environment variables
   - Регулярное обновление зависимостей
   - Security headers
